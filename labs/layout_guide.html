Overall Formatting

I made the body position relative. Minimum width of 1000 pixels and minimum height of 1000 pixels.

The unordered lists are displayed inline with font-size of 20px, top of 50px, right padding of 10px, and a thin solid black border of the right side of each item on the list except for the last one.

Site Header

The photo in the header has a static position.

Navigation Links

Site navigation links position is relative

Site Footer

Footer position is absolute. The width is set to 100 percent, height of 60 pixels, and zero for the bottom.
